== Description ==

Stiky Posts lets you show your sticky posts on your blog page.

= Main Features =
* **Multi-widget capable**. That is, you can have several widgets of Stiky Posts on your blog - each with its own settings!

== Installation ==

1. Download the plugin and extract its contents.
2. Upload the `sticky-posts` folder to the `/wp-content/plugins/` directory.
3. Activate **Stiky Posts** plugin through the "Plugins" menu in WordPress.
4. In your admin console, go to Appearance > Widgets, drag the Stiky Posts widget to wherever you want it to be and click on Save.

That's it!

= USAGE =

Stiky Posts can be used as a widget
